## What was implemented
1. Added new Python utility `tools/yc_bootstrap.py` that:
   - renders `infra/apigw.yaml` with function/service account IDs;
   - discovers API Gateway by name via YC API;
   - creates or updates API Gateway via YC API;
   - polls operation completion;
   - resolves and prints outputs in env format (`YC_API_GW_ID`, `YC_API_GW_NAME`, `YC_API_GW_DOMAIN`, `NOTIFY_ENDPOINT`);
   - supports `--dry-run` and `--output-env` modes.
2. Updated deploy workflow:
   - removed YC CLI install/auth/configuration steps;
   - added IAM token exchange step via SA JSON credentials;
   - replaced shell bootstrap call with `python tools/yc_bootstrap.py`;
   - exported bootstrap outputs into `$GITHUB_ENV` for smoke-check continuity.
